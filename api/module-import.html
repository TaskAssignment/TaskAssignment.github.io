<inner> readFile(modelName, transformCallback, headers, fileName)

Basic flow to read files. It's assumed that, whatever the file name, it's located on 'files/' (from the root of the project).
Parameters:
Name Type Default Description
modelName string The Model name that will be import.
transformCallback function The function that will transform the data before writing it to the database.
headers boolean | array true If true, it will consider the first line of the file as headers. If an array is given, each entry will be a header.
fileName String The name of file to be read. This should be a .tsv. Default is modelName pluralized. E.g: modelName = Issue, fileName = Issues.tsv

(inner) cosineSimilarity(nodesJson)

Calculate the cosine similarity based on the given bug and developer tags.
Parameters:
Name Type Description
nodesJson array Array with each of the tags, with their respective counters (bug, user, common). If the tag is a JSON, it's parsed to Object inside the function.

(inner) findMatches(req, res) → {Object}

Calculates all the similarity metrics (jaccard, cosine, SSA_Z) for a user and a bug.
Parameters:
Name Type Description
req Object Express request.
res Object Express response.
req.params.issueId Number The GitHubIssue id. This is necessary to filter all the candidate assignees.
Returns:
Sends a response (using res.send) with an Object containing the similarities of this issue.

(inner) mergeTags(modelsTags, callback, callbackParams)

Merge bug tags with user tags and call the given callback to handle them.
Parameters:
Name Type Description
modelsTags object Each model that will be merged is a key in this object. Tags must be under the name of the model they are associated with.
callback function The callback to handle the merged tags.
callbackParams object The params to pass to the callback. If empty, it will have the merged tags under the property 'tags'.

(inner) sort(value1, value2, username1, username2)

Descending order for the given values. If they are the same, sorts based on the usernames.
Parameters:
Name Type Description
value1 Number The first value of the comparison
value2 Number The second value of the comparison
username1 Number The username related to the first value.
username2 Number The username related to the second value.

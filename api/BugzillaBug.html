<inner> _id :String

The id on the source. Since this is a Bugzilla bug, this number should be unique on their database. However, since Bugzilla is an open source sofware, we use the service to assure the uniqueness of this id. E.g: If this bug comes from mozilla, it will have _id = 'MZ' + id.

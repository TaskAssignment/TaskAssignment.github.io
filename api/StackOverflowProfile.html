Class: StackOverflowProfile

StackOverflowProfile

new StackOverflowProfile()

This represents a user from StackOverflow (SO). It stores the tags, questions and answers related to this user on SO. If this user has an e-mail, it will be possible to gather information from multiple profiles and have a Developer profile.

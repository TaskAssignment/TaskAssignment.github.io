<inner> writeFile(modelName, headers, transform, fileName, items, filter, populate)

This is responsible for writing the file with the StackOverflow data. This is considered a 'private' method and all the write methods depend on it. All files generated with this method will be saved on the folder 'files/'.
Parameters:
Name Type Default Description
modelName String The Model that will be exported.
headers boolean | Array true If true, each attribute will be a header in the tsv file. If array, each entry will be a header and the other attributes will be ignored.
transform function The callback to transform the data. If none, the data will be saved the way it was read.
fileName String The name of the generated file. Default is the model name.
items String -updatedAt -__v The items that should be included or excluded when fetching the database.
filter Object The filter to the database query.
populate String The fields that will be populated when fetching from the database.

Class: Developer

Developer

new Developer()

This is the generic Developer profile. It gathers all the other profiles based on the user's email. One developer can have several profiles on different platforms, like GitHub and StackOverflow. Any other profiles must be referenced on this schema.
Requires:
  • module:mongoose
